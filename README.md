# FPS感度ツール スマホ版 UI最適化 v7

スマホ版は、かんたん設定を初期表示し、必要な時だけ詳細設定へ切り替える構成です。

- 2本指基準
- かんたん設定 / 詳細設定の明確な切替
- 武器ごとの主軸スコープに自分の感度を入力
- DMR / SR は4倍・6倍・8倍基準
- 肩越しTPP / FPP対応
- ジャイロON/OFF対応
- PWA対応

注意：本ツールは、実戦向けの参考感度を算出する非公式ツールです。最終的な感度は端末・指配置・プレイスタイル・ジャイロの使い方・個人の感覚によって変わります。


## v7変更点
- かんたん設定から武器選択を削除しました。
- かんたん設定は「ジャイロあり / ジャイロなし」だけで全武器共通の目安を表示します。
- 武器別に細かく合わせる場合は詳細設定で行います。
